# 变更日志

本文记录 OpenDocX 的公开版本变化。

## v0.1.0-alpha - 2026-06-08

首个公开 alpha 版本，目标是提供一条完整可跑通的主链路：项目文档管理、AI 辅助编辑、发布确认、静态站构建、反馈审核和审计。

### 新增

- 项目、版本和文档树管理。
- Markdown 三栏编辑器。
- AI 浮层与 AI 面板。
- 发布前确认弹窗。
- 静态站构建。
- 反馈提交与审核。
- 用户管理和审计日志。
- Demo seed 数据。
- Docker Compose 部署入口。
- 中文 README、用户指南、架构说明、路线图、API 参考。
- Markdown 渲染能力示例文档。

### 静态站渲染

- 支持常见 Markdown 语法。
- 支持表格、任务列表、引用、提示块。
- 支持 Pygments 代码高亮。
- 支持图片、GIF 和视频链接展示。
- 支持浅色/深色主题。
- 支持 Mermaid 与公式的容器级样式。

### 安全

- 默认不提交 `.env`、本地数据库和构建产物。
- Demo 凭据仅用于本地试用。
- Markdown 默认转义并清洗。
- 管理员路由需要认证。
- 关键操作写入审计日志。

### 已知不足

- 前端测试仍需补齐。
- 静态站搜索与 SEO 仍需增强。
- 媒体上传和资产管理仍在路线图中。
- 多人实时协作尚未实现。
- MDX 尚未支持。
